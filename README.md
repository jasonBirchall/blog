# blog

A personal linkblog and hub with essays, links, quotes,
TILs, and notes — built on Django 5 and SQLite, managed with `uv`. Markdown
files in `content/` are the source of truth; the database is a derived
artefact. Canonical home is Codeberg; GitHub holds a read-only mirror. Code is
licensed MIT, prose and other content under CC BY-SA 4.0 — see `LICENSE`.
